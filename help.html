An unofficial JSON API for The Onion's Horoscopes

Welcome, dear visitor!

If you, like me, enjoy the hilarity and witty satire of The Onion's Horoscopes, but have been looking for a (forgiving) JSON API, look no further!

I've exposed a couple of resources, all accessible via GET requests.

http://a.knrz.co/horoscope-api/current
will return an array of all the latest horoscopes.

http://a.knrz.co/horoscope-api/current/:sign
where :sign is your astrological sign (example, gemini), will return just the horoscope for that sign.

http://a.knrz.co/horoscope-api/archive/:year/:week
where :year and :week are self-explanatory (example, 2015 and 32). This will return an array of the last published horoscopes before that week.

http://a.knrz.co/horoscope-api/archive/:year/:week/:sign
the same as the previous one, but it will only return the horoscope for that week.

How it works
By default, any request for a particular week (even the current week) will always return horoscopes, regardless of whether or not The Onion actually published horoscopes for that week. For example, requesting the horoscopes for the 39th week of 2015 (for which The Onion did not publish anything) will return the horoscopes for the 37th week, as those were the last ones published. If you would like to be strict in your requests, for whatever reason, add ?human=false to the end of your request.

If you, dear visitor, are from The Onion and believe that I've overstepped, I do apologize. It was unintentional. Please contact me so we can come up with a resolution, even if it means taking this service down.
